----- Iniciar Git ---

1. Iniciar um Novo Repositório Git no Projeto
1.	Abra o VS Code e navegue até o diretório do seu projeto XXXXX.
2.	Abra o terminal integrado no VS Code (Ctrl + `) e execute o seguinte comando para inicializar um novo repositório Git:

git init

Isso cria um novo repositório Git na pasta atual.

2. Adicionar os Arquivos ao Controle de Versão

git add .

Isso adiciona todos os arquivos do diretório do projeto ao estágio de commit.

2.	Faça o commit inicial dos arquivos:

git commit -m "Commit inicial do projeto XXXXX"

3. Criar um Novo Repositório no GitHub

1.	Acesse GitHub e faça login na sua conta.
2.	Crie um novo repositório:
o	Clique no ícone de "+" no canto superior direito e selecione "New repository".
o	Dê um nome ao seu repositório (por exemplo, spedpyutils).
o	Escolha se o repositório será público ou privado.
o	Não adicione nenhum arquivo README, .gitignore, ou licença por enquanto, pois você já tem esses arquivos localmente.
o	Clique em "Create repository".

4. Conectar o Repositório Local ao GitHub

1.	No terminal do VS Code, conecte seu repositório local ao novo repositório GitHub:

git remote add origin https://github.com/seu_usuario/XXXXX.git

Substitua seu_usuario pelo seu nome de usuário no GitHub.

2.	Envie os commits locais para o repositório remoto no GitHub:

git push -u origin master

5. Verificar o Repositório no GitHub
•	Acesse o repositório no GitHub para confirmar que todos os arquivos foram enviados corretamente.
